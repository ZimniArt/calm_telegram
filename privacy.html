Privacy Policy

Telegram Calm is a browser extension that modifies the appearance of the Telegram Web interface on the user's device. It hides subscriber, member, and view counters according to the user's preference.

Data Collection

The extension does not collect, store, transmit, or share any personal or usage data. No information is sent to external servers.

Local Storage

The only data stored is a single ON/OFF toggle value, saved locally using the browser's storage API. This value is used solely to control the extension's behavior.

Permissions

The extension uses permissions only to modify the Telegram Web page content in the browser. It does not access any other websites or data.

Contact

If you have questions, you may contact the developer through the Chrome Web Store contact form.
